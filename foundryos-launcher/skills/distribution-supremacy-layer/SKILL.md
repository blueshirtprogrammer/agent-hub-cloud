# Distribution Supremacy Layer Skill

## Purpose

Use this skill when FOUNDRYOS needs to become installable, purchasable, and discoverable across many distribution surfaces.

## Core Loop

```text
Select channel
Package product
Prepare listing
Check requirements
Create launch assets
Publish or queue approval
Measure adoption
Improve channel pack
```

## Channels

- WordPress plugin
- WordPress theme/block kit
- npm package
- Docker image
- AWS Marketplace
- HighLevel Marketplace app
- Product Hunt
- affiliate program
- white-label agency app
- skill marketplaces
- GitHub releases
- enterprise private deployment

- buyer access
- speed to launch
