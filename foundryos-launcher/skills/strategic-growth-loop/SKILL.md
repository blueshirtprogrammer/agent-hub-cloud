# Strategic Growth Loop Skill

## Purpose

Use this skill when FOUNDRYOS needs to think for itself, set its own next goal, and improve without waiting for the owner to prompt it.

## Core Loop

```text
Observe
Dream
Set Goal
Prioritise
Execute
Measure
Learn
Improve Skill
Update Strategy
Repeat
```

## Daily Protocol

1. Read current company state.
2. Identify the current monetisation stage.
3. Find the largest bottleneck.
4. Dream three possible opportunities.
5. Pick one goal for today.
6. Design one small experiment.
7. Assign the right agent or department.
8. Define proof of completion.
9. Measure result.
10. Recommend one skill or process improvement.
11. Write a short board summary.

## Goal Types

- revenue goal
- product goal
- launch goal
- distribution goal
- customer proof goal
- skill improvement goal
- licensing goal
- case study goal
- valuation goal
- exit readiness goal

```markdown
# Weekly Dream Memo

## What we learned

## What is working

```

## Rule

No dreaming without execution.

Every dream must produce at least one of:

- goal
- experiment
- task
- skill update
- product improvement
- proof asset
- revenue action
- strategy change
