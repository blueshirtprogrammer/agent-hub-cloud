Embed the FOUNDRYOS AI company launcher, founder intake, templates, and deal-room entry points into WordPress.

== Description ==

FOUNDRYOS Launcher lets WordPress site owners, agencies, consultants, and venture studios embed a FOUNDRYOS company launcher into their website.

Shortcodes:

* [foundryos_launcher]
* [foundryos_founder_intake]
* [foundryos_deal_room]

This plugin connects to your FOUNDRYOS SaaS endpoint. Premium templates, self-launch autopilot, licence heartbeat, and company lease features require an active FOUNDRYOS licence.

== Installation ==

1. Upload the plugin folder to /wp-content/plugins/.
2. Activate the plugin.
3. Go to Settings > FOUNDRYOS Launcher.
4. Add your FOUNDRYOS endpoint and licence key.
5. Add [foundryos_launcher] to a page.

== Frequently Asked Questions ==

= Does this plugin include the full FOUNDRYOS runtime? =

No. It embeds and connects to your FOUNDRYOS runtime or SaaS endpoint.

= Can agencies white-label it? =

Yes. White-label mode is included as a setting for agency deployments.

= Does it collect data? =

Only data intentionally submitted through the embedded FOUNDRYOS experience, subject to the configured FOUNDRYOS endpoint and privacy policy.
