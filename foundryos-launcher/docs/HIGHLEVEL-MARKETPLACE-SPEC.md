# FOUNDRYOS HighLevel Marketplace Spec

## Purpose

Turn FOUNDRYOS into a white-label agency app for the HighLevel ecosystem.

## App concept

```text
FOUNDRYOS Agency Company Launcher
```

## Buyer

```text
GHL agency owners
automation consultants
AI agencies
SaaSpreneurs
funnel builders
white-label software operators
```

## Core features

- install into agency or sub-account
- OAuth app install
- agency white-label support
- founder install pipeline
- company template launcher
- lead capture form
- opportunity creation
- proposal workflow
- payment link handoff
- client template installs
- licence heartbeat status

## OAuth notes

Start as Private App for development and testing.

Move to Public App only when stable and secure.

Recommended target:

```text
Both Agency and Sub-account where appropriate
```

Recommended listing type:

```text
White-label friendly
```
