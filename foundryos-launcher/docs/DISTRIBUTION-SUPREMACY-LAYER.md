## Distribution surfaces

## 1. WordPress plugin

Goal:

Make FOUNDRYOS accessible to the biggest non-technical business website market.

Plugin concept:

```text
FOUNDRYOS Launcher for WordPress
```

Functions:

- embed company launcher widget
- connect licence key
- display founder install funnel
- show templates
- collect venture idea submissions
- route to FOUNDRYOS deal room
- expose partner/affiliate CTA
- optional white-label agency mode

Submission guardrails:

- GPL-compatible plugin code
- clear SaaS disclosure
- consent for tracking
- human-readable code
- no dashboard hijacking
- no deceptive upsells
- no hidden external code execution

## Distribution scorecard

Score each channel 1-5:

- buyer access
- speed to launch
- technical complexity
- approval complexity
- revenue potential
- recurring revenue potential
- partnership leverage
- brand visibility
- support burden
- strategic value

Verdict:

```text
0-20: skip
21-30: research
31-40: package
41-50: launch now
```
