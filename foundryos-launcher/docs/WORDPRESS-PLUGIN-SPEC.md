# FOUNDRYOS WordPress Plugin Spec

## Name

```text
FOUNDRYOS Launcher
```

## Purpose

Let WordPress site owners, agencies, consultants, and venture studios embed a FOUNDRYOS company launcher, founder intake form, template browser, and licence-connected SaaS experience into their website.

## Core features

- Admin settings page
- Licence key field
- FOUNDRYOS API endpoint field
- Launcher embed shortcode
- Gutenberg/block embed
- Founder idea intake form
- Template browser
- Deal room CTA
- Affiliate tracking field
- White-label agency mode
- Privacy disclosure
- Support link

## Shortcodes

```text
[foundryos_launcher]
[foundryos_founder_intake]
[foundryos_templates]
[foundryos_deal_room]
```
