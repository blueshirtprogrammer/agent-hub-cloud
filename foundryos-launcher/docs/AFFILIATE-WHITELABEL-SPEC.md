# FOUNDRYOS Affiliate and White-Label Spec

## Purpose

Turn agencies, consultants, creators, and operators into FOUNDRYOS distribution partners.

## Partner types

```text
Affiliate
Agency reseller
White-label agency
Enterprise referral partner
Venture studio partner
Implementation partner
```

## Commission ideas

```text
Self-install template: 20-40 percent
Founder install referral: fixed fee or percentage
Monthly company lease: recurring rev share
Enterprise intro: success fee
White-label agency: wholesale/retail margin
```

## Affiliate offer

```text
Refer founders and agencies to FOUNDRYOS Founder Install.
```

## White-label offer

```text
Agency AI Company Launcher
```

Includes:

- agency-branded launcher
- GHL pipeline
- client template installs
- licence heartbeat
- revenue autopilot
- case study workflow
- marketplace listing support

## Partner onboarding assets

- partner landing page
- demo video
- one-pager
- swipe copy
- founder install script
- comparison table
- FAQ
- payout terms draft

## Partner scorecard

Score 1-5:

- audience fit
- trust
- deal flow
- technical capability
- support burden
- expected referrals
- strategic value

## Rule

Do not give deep product control to low-trust partners.

Start with referral, then reseller, then white-label only after proof.
