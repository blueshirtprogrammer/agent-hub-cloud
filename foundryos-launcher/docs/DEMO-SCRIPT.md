# FOUNDRYOS™ Demo Script

## Demo goal

Show that FOUNDRYOS is not an AI assistant.

It is an installer for AI companies.

## Demo length

```text
5-7 minutes
```

## Opening line

> Everyone is building AI employees. We are building AI companies.

## Scene 1: The problem

Say:

> AI tools are fragmented. Founders have coding agents, browser agents, model routers, CRMs, billing tools, docs, deployment platforms, and no company structure tying it together.

Show:

- Launcher homepage.
- Install sequence.
- Connector cockpit.

## Scene 2: The thesis

Say:

> FOUNDRYOS turns a business idea into an installable company package: CEO, C-suite, departments, revenue loop, approval gates, proposal/invoice workflows, and proof-of-work reporting.

Show:

- Company template picker.
- Default template: Enterprise Hospitality AI Company.

## Scene 3: The wedge

Say:

> We start narrow: enterprise hospitality AI. Hotels leak revenue across concierge, minibar, procurement, venue upsell, and supplier activation. The company template packages an AI firm around that pain.

Show:

- Template description.
- Mention AI concierge, smart minibar, procurement automation, venue CRM, supplier dashboard.

## Scene 4: The install plan

Say:

> The launcher generates an install plan: connect credentials, import company, install skills, deploy runtime, start the first revenue loop.

Click:

```text
Generate install plan
```

Show:

- API response or JSON plan.

## Scene 5: The company operating loop

Say:

> The first loop is not vague planning. It is CEO to CRO to RevOps to proposal/invoice to delivery to board report.

Show:

```text
CEO → CRO → RevOps → Proposal/Invoice → COO Delivery → Board Report
```

## Scene 6: Trust layer

Say:

> The reason this can sell to serious businesses is governance. External sending, invoices, contracts, public claims, production deletes, and capital raise material stay approval-gated.

Show:

- Approval gates.
- Connector docs.
- Security guardrails.

## Scene 7: Revenue path

Say:

> The first thing we sell is not unlimited software. It is a paid founding install. We deploy the user’s first AI company package and configure its first revenue sprint.

Show:

```text
FOUNDRYOS Founder Install: $2,500
```

## Scene 8: Close

Say:

> This is the shift: from AI tools that help with tasks, to AI companies that can be installed, operated, measured, and improved.

Final line:

> FOUNDRYOS does not generate business plans. FOUNDRYOS deploys companies.

## Demo CTA

For agencies:

> Want us to deploy your first AI company template around your niche?

For operators:

> Want to package your business model into an AI-run company shell?

For enterprise:

> Want a governed internal AI company with approval gates and proof-of-work reporting?
