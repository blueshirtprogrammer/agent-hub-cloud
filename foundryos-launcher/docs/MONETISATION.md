## Price ladder

```text
Free: demo launcher and local company template
$297-$997: self-install templates
$1,500-$5,000: done-with-you setup
$499-$2,500/month: company lease / managed runtime
$5,000-$25,000/month: enterprise private foundry
```

## Marketplace entitlement model

A user can install a company package only if they have one of:

- active subscription,
- one-time template purchase,
- enterprise license,
- internal founder/admin override.
