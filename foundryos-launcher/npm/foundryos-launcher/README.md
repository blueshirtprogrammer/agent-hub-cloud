# @foundryos/launcher

npm package scaffold for the FOUNDRYOS Launcher ecosystem.

## Goal

Make FOUNDRYOS installable from the command line for developers, agencies, and venture studios.

## Intended commands

```bash
npx @foundryos/launcher init
npx @foundryos/launcher install-company
npx @foundryos/launcher install-skills
npx @foundryos/launcher status
npx @foundryos/launcher spawn
npx @foundryos/launcher deal-room
npx @foundryos/launcher war-room
```

## Current status

This package is a scaffold. It exports the command map and product doctrine. The executable CLI should be wired after the runtime, licence heartbeat, and package installer are finalised.

## Doctrine

Deploy AI companies, not AI assistants.
